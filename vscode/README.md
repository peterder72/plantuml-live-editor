# PlantUML Live Preview

Open a PlantUML document and run **PlantUML: Open Preview to the Side**.

The preview renders entirely inside VS Code without a PlantUML server, Java
runtime, CDN, or network access. It includes pan, zoom, fit, reset, SVG/PNG
export, and live boolean toggles declared as:

```plantuml
!$_live_SHOW_DETAILS = false
```
